# Genomic Sequence Analysis Using BioPython

## Overview
This project analyzes DNA sequence data using BioPython to compute GC content and visualize nucleotide composition across a genomic sequence.

## Tools Used
- Python
- BioPython
- Matplotlib
- NCBI Entrez database

## Workflow
1. Fetch DNA sequence from NCBI
2. Compute GC content using sliding window analysis
3. Perform basic sequence alignment
4. Visualize GC distribution using matplotlib

## Key Output
- GC content variation plot across the genome
- Sequence statistics (length, composition)

## Biological Significance
GC-rich regions are often associated with gene-dense areas and regulatory elements in DNA.
